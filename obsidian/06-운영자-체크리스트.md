## 운영 방침 (결정됨)

100명 이하 규모 → **홈페이지 관리 기능 없이 Google Sheets로 운영**

| 하지 않음 | 대신 |
|-----------|------|
| 운영자 대시보드 | 시트 + 필터 |
| 참가자 로그인 | 없음 |
| 셀프 취소/코스 변경 | 카카오 문의 |
| 자동 중복 DB 차단 | 시트에서 수동 확인 |

## 시트 운영

### 추천 열 (확장)

기본 7열 + 아래 추가 권장:

| 열 | 용도 |
|----|------|
| 지역 | locationLabel |
| 선호 요일 | runningDaysLabel |
| 선호 시간 | runningTimePreference |
| 입금확인 | Y/N (수동) |
| 중복체크 | 운영자 메모 |
| 톡방 연결 | Y/N/희망 |
| 발송완료 | Y/N |

### 일일 운영 루틴 (모집 중)

- [ ] 새 신청 행 확인
- [ ] 입금 대조 (이름·금액)
- [ ] 입금확인 열 업데이트
- [ ] 동일 이름·전화 중복 여부 확인
- [ ] 문의(카톡) 응대

### 중복·부정 수령 방지 (수동)

1. 신청 시: 이름 + 휴대폰으로 기존 행 검색
2. 다른 코스 추가 신청 → 카톡으로 안내 후 1코스만 유지
3. 입금자명 ≠ 신청자명 → 확인 연락
