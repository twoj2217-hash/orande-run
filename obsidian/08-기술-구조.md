| 계층 | 기술 |
|------|------|
| 프레임워크 | Next.js 15 (App Router) |

## 페이지 라우트

| URL | 설명 |
|-----|------|

| `POST /api/apply` | 신청 제출 |

## 핵심 설정 파일

`lib/event-config.ts`에서 관리하는 것:

- `brandCopy` — 브랜드 문구
- `eventSchedule` — 일정
- `runTiers` — 코스·가격·리워드
- `paymentInfo` — 계좌
- `verificationConfig` — 인증
- `participantGuide` — 완료 후 안내
- `getRecruitmentStatus()` — 모집 상태

**운영 값 변경 시 이 파일만 수정하면 대부분 반영됨.**

## 로컬 개발

```bash
npm install
npm run dev    # http://localhost:3000
```

환경변수 없이도 UI 확인 가능. 실제 신청 제출은 `.env.local` 필요.
