# OranDe Run 발송 지연 공지 템플릿

작성일: 2026-06-29

## 1) 오픈채팅 공지 템플릿 (짧은 버전)

안녕하세요, 오랜디런 운영팀입니다.  
완주 리워드 발송 일정이 일부 지연되고 있어 먼저 안내드립니다.

- 기존 안내: 챌린지 종료 후 2주 이내
- 현재 상황: `[지연 사유 입력]`
- 업데이트 예정일: `[YYYY-MM-DD]`

불편을 드려 죄송합니다.  
개별 문의는 오픈채팅으로 남겨 주시면 순차적으로 답변드리겠습니다.

## 2) 상세 공지 템플릿 (DM/공지글)

안녕하세요, 오랜디런 운영팀입니다.

리워드 발송이 지연되어 안내드립니다.

1. 지연 사유  
   - `[예: 제작사 일정 지연 / 주소 재확인 필요 / 택배 집하 지연]`

2. 현재 진행 상황  
   - `[예: 포장 완료율 70%, 1차 출고 완료]`

3. 다음 안내 시점  
   - `[YYYY-MM-DD HH:MM]`

4. 문의 채널  
   - 카카오 오픈채팅 `OranDe Run 문의`

운영 지연으로 불편을 드려 죄송합니다.  
변경 사항은 동일 채널에 즉시 업데이트하겠습니다.
