# OranDe Run 운영자 대응 체크리스트

작성일: 2026-06-29  
적용 범위: 모집 전 ~ 발송 완료

## 1) 모집 전 (오픈 전 3일)

### 필수 확정 항목

- [ ] 모집 기간/참여 기간/인증 마감일 공지 일치 확인
- [ ] 입금 기한 확정 및 안내 문구 반영
- [ ] 입금 후 취소/환불 기준 확정 및 안내 문구 반영
- [ ] 완주 인정 기준(거리/기간/증빙) 확정
- [ ] 비공개 인스타 인증 대안(DM) 확정
- [ ] 톡방 연결 운영 여부 확정 (미실시 시 문구도 확정)
- [ ] 발송 지연 시 공지 기준(예: 3일 초과) 확정

### 채널 준비

- [ ] 카카오 오픈채팅 고정 메시지 등록
- [ ] 인스타 고정 게시물/하이라이트 최신 일정 반영
- [ ] 당근 공지 문구를 과도한 상업 홍보 톤 없이 최소 안내 형태로 작성

### 운영 자료 준비

- [ ] 참가자 FAQ 최신본 배포
- [ ] 분쟁 답변 템플릿 3종 준비(환불, 완주 미달, 발송 지연)
- [ ] 시트 열 구조/필터/정렬 기준 점검

## 2) 모집 중 (일일 루틴)

## D1. 오전(09:00~11:00)

- [ ] 신규 신청 건 확인
- [ ] 입금 내역 대조
- [ ] 동일 이름/전화 중복 신청 확인
- [ ] 입금자명 불일치 건 별도 라벨 처리

## D2. 오후(14:00~16:00)

- [ ] 입금확인 열 업데이트 (`Y/N/확인중`)
- [ ] 문의 미응답 24시간 초과 건 우선 처리
- [ ] 취소/변경 요청 처리 후 로그 기록

## D3. 저녁(20:00~21:00)

- [ ] 카카오/인스타 문의 마감 공지
- [ ] 당일 이슈 요약 기록 (분쟁성/지연성/정책 모호성)
- [ ] 다음날 공지 필요 항목 정리

## 3) 모집 마감 직후 ~ 참여 시작 전

- [ ] 최종 참가 확정 명단 확정(입금 기준)
- [ ] 미입금자 최종 안내 후 상태 정리
- [ ] 참여 시작 D-3, D-1 리마인드 공지 발행
- [ ] 인증 방법 재공지(해시태그/DM 둘 다)
- [ ] 문의방 안내 문구에서 "참가 확정 기준" 재강조

## 4) 챌린지 진행 중 (주간 루틴)

## W1. 주 1회 공지

- [ ] 진행 독려 공지(부담 없는 톤)
- [ ] 인증 예시 이미지 1~2개 공유
- [ ] 자주 묻는 질문 상단 고정

## W2. 운영 확인

- [ ] 중도포기/부상 문의 분리 라벨링
- [ ] 코스 변경 요청 건 정책 기준으로 일괄 처리
- [ ] 톡방 연결 요청 현황 점검(미실시면 안내 문구 유지)

## 5) 인증 마감 주간 (8/10~8/16)

### 인증 접수 체크리스트

- [ ] 이름/코스/날짜/기록 총거리 확인
- [ ] 기한 내 제출 여부 확인
- [ ] 비공개 계정은 DM 인증으로 처리
- [ ] 판정 근거를 시트 메모에 1줄 기록

### 마감 운영 체크

- [ ] 마감 D-2, D-1 긴급 리마인드 공지
- [ ] 예외 요청(지연 제출) 기준 통일
- [ ] 판정 결과 공지 템플릿 사용

## 6) 발송 준비 ~ 발송 완료

### 발송 전

- [ ] 수령지 재확인(우편번호/상세주소)
- [ ] 제작 수량과 완주 확정 수량 대조
- [ ] 제작 지연 시 사전 공지 발행

### 발송 후

- [ ] 발송완료 열 업데이트
- [ ] 미수령/반송 건 라벨링
- [ ] 재발송 정책에 따라 처리

## 7) 시트 열 업데이트 순서 (권장)

1. `입금확인` 업데이트  
2. `중복체크 메모` 업데이트  
3. `참가확정` 상태 확정  
4. `완주인증` 판정 기록  
5. `발송완료` 업데이트

## 8) 긴급 상황 대응 분기

### A. 모집 즉시 중단 필요

1. 모집 상태 차단  
2. 상단 배너/문의방 공지 즉시 발행  
3. 접수 중 건 환불/유예 기준 공지

### B. 시트 연동 오류

1. API 상태 확인  
2. 환경변수/웹훅 URL/시크릿 재점검  
3. 임시 수기 접수 폼 또는 카카오 접수 대체 공지

### C. 굿즈 발송 지연

1. 예상 지연 일수 산정  
2. 공지 템플릿으로 사전 안내  
3. 문의 대응 문구 통일

## 9) 운영 품질 KPI (간단 버전)

- 입금 확인 평균 리드타임: 24시간 이내
- 문의 첫 응답 리드타임: 12시간 이내
- 인증 판정 리드타임: 48시간 이내
- 발송 지연 공지 누락: 0건
