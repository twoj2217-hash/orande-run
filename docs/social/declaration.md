# OranDe Run 선언문 (고정 게시물 + 카드뉴스)

오랜디런 고정 게시물용 선언문입니다.

## 1) 카드뉴스 구성 (7장)

## 카드 1 - 표지

- 제목: `오랜디런 선언문`
- 서브: `대전 로컬 · 제1회`
- 하단 로고: `OranDe Run`

## 카드 2 - 왜 버추얼 런인가

우리는 한곳에 모이지 않아도 달릴 수 있다고 믿어요.  
날씨, 시간, 장소와 상관없이 내 일정에 맞게 뛰는 게 진짜 러닝이에요.

## 카드 3 - 누구나 참여

러닝머신이든 공원이든, 처음이든 오래 달린 사람이든 괜찮아요.  
오랜디런에는 "내 페이스"가 가장 중요해요.

## 카드 4 - 함께 달리기

혼자 달려도 좋고, SNS로 응원을 나눠도 좋아요.  
`#오랜디런` `#OranDeRun` 으로 서로의 기록을 응원해요.

## 카드 5 - 리워드

뻔한 메달 대신, 오랜디런만의 키캡키링을 받아요.  
50km 2구 · 70km 3구 · 100km 4구로 완주의 증거를 남겨요.

## 카드 6 - 일정

모집 7.1 ~ 7.5  
4주 챌린지 7.13 ~ 8.9  
완주 인증 마감 8.16

## 카드 7 - CTA

이번엔 내 페이스로, 4주 버추얼 런.  
프로필 링크에서 참가 신청해요.

해시태그:
`#오랜디런` `#OranDeRun` `#버추얼런` `#대전러닝` `#러닝챌린지`

## 3) 카드뉴스 제작 규격

- 크기: 1080 x 1350 (4:5)
- 색상: 크림 배경 + 오렌지 포인트 + 차콜 텍스트
- 폰트: Pretendard Black(제목), Pretendard Medium(본문)
- 금지: 보라색 톤, blur 원 장식, 과한 3D 질감
- 공통 요소: 하단에 작은 `OranDe Run` 표기 + 페이지 번호
