# OranDe Run 인스타그램 프로필 세팅안

OranDe Run(오랜디런) 브랜드 톤에 맞춘 인스타그램 프로필 운영 초안입니다.

## 1) 계정 아이디 후보

1순위부터 확인해 주세요.

- `@oranderun`
- `@orande.run`
- `@oranderun_official`
- `@oranderun_daejeon`

## 2) 프로필 기본값

### 이름 (30자 이내)

`OranDe Run | 오랜디런`

### 소개글 (150자 이내)

아래 문구를 그대로 사용해도 됩니다.

```text
대전 제1회 비대면 버추얼 런
누구나, 내 페이스로 4주 달리기
완주하면 키캡키링
모집 7.1~7.5 · 참여 7.13~8.9
참가 신청 ↓
```

### 카테고리

- 커뮤니티
- 스포츠

### 링크인바이오

- 링크 텍스트: `참가 신청하기`
- 랜딩 URL: `https://oranderun.vercel.app` (임시, 배포 도메인 확정 시 교체)

## 3) 프로필 이미지 가이드

둘 중 1개 선택:

- 키캡키링 포스터 크롭(원형)
- 오렌지 배경 + `OranDe Run` 워드마크

권장 규격:

- 원본 작업: 1080 x 1080
- 업로드용: 320 x 320 이상

## 4) 하이라이트 구성 (5종)

| 하이라이트 | 설명 |
|---|---|
| 선언문 | 오랜디런 운영 철학 요약 |
| 참여방법 | 신청 → 4주 러닝 → 인증 → 발송 |
| 코스안내 | 50km / 70km / 100km + 참가비 |
| 인증방법 | 해시태그 또는 인스타 DM 인증 |
| 문의 | 카카오 오픈채팅 연결 |

## 5) 운영 메모

- 말투는 `~해요` 중심으로 통일합니다.
- 딱딱한 표현(예: 온라인 마라톤 대회)은 피합니다.
- 핵심 해시태그는 `#오랜디런` `#OranDeRun`를 고정으로 사용합니다.
