# OranDe Run · 디자인 가이드

## 브랜드 한 줄

비대면 버추얼 런, 누구나 달리고 키캡키링을 모으는 챌린지.

## 핵심 메시지

1. **버추얼 런 / 비대면 마라톤** — 현장 집결 없이 각자 달림
2. **누구나 참여** — 러닝머신·공원·어디서든, 실력 제한 없음
3. **리워드 = 키캡키링** — 20대 사이 유행 굿즈, 코스별 2구/3구/4구

## 용어 규칙

| 쓰는 말 | 쓰지 않는 말 |
|---------|-------------|
| 키캡키링 | 키캡 + 키링 (별도 품목) |
| 키캡키링 2구 | 키캡 1 + 키링 1 |
| 버추얼 런, 비대면 마라톤 | 온라인 마라톤 대회 (딱딱한 표현) |

대전 로컬·제1회 → 히어로 뱃지·푸터. 당근 모임 출처 → 푸터·메타만.

## UI 분류

- **Landing:** 히어로, CTA — 브랜드·감정 우선, 카드 남용 금지
- **App UI:** 비교표, 신청 폼 — 정보 밀도·가독성 우선

## 컬러

- 액센트: `--primary` (오렌지)
- 배경: 크림 `--background`
- 텍스트: 차콜 `--foreground`
- 그라데이션은 히어로 오버레이 1곳만. 섹션 배경 blur 원·purple 계열 금지

## 타이포

- 본문·제목: Pretendard
- 디스플레이: `font-black` + `tracking-wider`
- 한글: `text-ko-balance` 클래스 사용

## 모션 (최대 3종)

1. 히어로 진입 (또는 스크롤 힌트)
2. CTA/타임라인 scroll-linked
3. 버튼 hover

`prefers-reduced-motion: reduce` 시 모션 비활성.
